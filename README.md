# 使い方

```
git clone https://github.com/DogKingFun/public-vue.git
cd public-vue
docker-compose up --build
```

　クローンをしてからdocker-composeを立ち上げるだけ！
　docker-composeがないなら入れようね。
  
　立ち上げれば、そのVPSサーバのIPアドレスにhttp通信できる。
   
http:// IP:80
